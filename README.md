# nini's Magical Journey

Personal static blog rebuilt from the old Hexo-generated export.

## Local build

```bash
npm run build
```

The source posts live in `src/posts`. Running the build script regenerates the
home page, archives, categories, tags, and article pages.

## GitHub Pages note

This repository is `lbenny123/luobeini.github.io`, so GitHub treats it as a
project site unless the repository is renamed/transferred or a custom domain is
configured. The default project-page URL is:

```text
https://lbenny123.github.io/luobeini.github.io/
```

For a root user-site URL under the `lbenny123` account, the repository must be
named `lbenny123.github.io`.
