关于这个小站

Becoming is not my concern. Leaving traces, that's enough.

这里保留的是一些生活化的痕迹：看完一部电影后的余震，某一周没来得及展开的日记，还有把技术问题从混乱里理出来的一点点快乐。

旧 Hexo 静态页面已经迁移成 8 篇文章源文件。之后新增内容时，只要在 src/posts 里加一篇带 frontmatter 的 HTML，再运行 npm run build 即可生成新站。
